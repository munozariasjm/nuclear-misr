# Nuclear PyModels

A simple python package to call naive python models

## Installation

First clone the repository

```bash
git clone ...
```

Then install the package

```bash
cd nuclear-pymodels
pip install .
```

## Usage

```python

from nuclear_pymodels import NuclearModel
... WIP

```
